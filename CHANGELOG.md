# Changelog

All notable changes to MedCabinet AI should be documented here.

## Unreleased

### Added
- Complete project documentation covering product requirements, roadmap, architecture, database design, API design, AI behavior, scanner flows, barcode data, shopping assistance, price comparison, UI/UX, testing, security, and contribution standards.
- GitHub write-access verification file.

### Current prototype capabilities
- Supabase-backed medicine inventory and photo storage
- AI-assisted medicine package scanning through a Netlify function
- Expiry, low-stock, category, location, dosage, price, store, and barcode tracking
- Cabinet search and filters
- Favorites and deleted-item waste history
- Cabinet health score and PDF/QR report sharing
- Configurable home currency with THB as default

## Documentation policy
Future product, schema, security, and user-facing behavior changes should update this file and the relevant document under `docs/` in the same pull request or commit.
